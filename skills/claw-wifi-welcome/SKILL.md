# 🦞 龙虾WiFi (Claw WiFi) 功能清单

欢迎使用龙虾WiFi！我是您的 AI 网络专家。我具备管理企业级路由器、优化无线网络、部署全球组网以及提升商业运营能力的多种技能。

以下是您可以探索的功能分类及推荐使用方式：

---

### 1. 🛡️ 路由器基础管理 (ClawWRT)
管理您的路由器硬件、客户端及 WiFi 设置。
- **核心功能**: 查看设备状态、WiFi 名称密码修改、踢出蹭网设备、实时流量限速。
- **推荐 Prompt**:
  - *"帮我看看现在有哪些路由器在线，运行得怎么样？"*
  - *"把 WiFi 名称改成 'Claw-Guest'，密码设为 '12345678'。"*
  - *"现在的实时流量峰值是多少？谁在占用宽带？"*

### 2. 🎨 商业门户页设计 (Portal Pages)
为您的场馆（咖啡厅、酒店、办公室）生成商用级认证页面。
- **核心功能**: 现代玻璃拟态 UI 生成、多套行业模板（咖啡、商务、酒店等）、自定义品牌 Logo 与文案。
- **推荐 Prompt**:
  - *"帮我为 '瑞幸咖啡' 生成一个温馨的咖啡主题认证页，并发布到路由器。"*
  - *"创建一个需要输入口令才能上网的欢迎页，背景要动态流动的。"*

### 3. 🌐 全球组网与 VPN (WireGuard)
在 VPS 与多台路由器之间建立安全的局域网隧道。
- **核心功能**: 跨地域内网互通、域名智能分流（如 Google/Telegram 走 VPN）、VPS 中转组网。
- **推荐 Prompt**:
  - *"帮我把北京和上海的两个办公室组个网，让两边的内网可以互访。"*
  - *"设置一下 VPN 路由，让所有访问 github.com 的流量都走隧道。"*

### 4. 🚀 内网穿透 (Intranet Penetration)
无需公网 IP，即可随时随地访问内网服务。
- **核心功能**: 自动部署内网穿透服务端、映射本地服务端口到公网。
- **推荐 Prompt**:
  - *"帮我在 VPS 上安装内网穿透服务端并设为自启动，然后把路由器的 SSH 22 端口转发到 6022。完成后，请帮我检查 VPS 上的 6022 端口是否已经正常开启监听。"*
  - *"帮我在 VPS 上搭个内网穿透服务端，把路由器的 80 端口映射出去。"*
  - *"我想在外网访问家里的 NAS，能帮我配一下内网穿透吗？"*

### 5. 📢 社交与活跃 (Moltbook)
让您的路由器在 Agent 网络中展现自我，分享运行状态。
- **核心功能**: 生成酷炫的网络运行报告、在 Moltbook 社区发帖。
- **推荐 Prompt**:
  - *"在 Moltbook 上发个贴，秀一下我们今天服务的上千名用户和极速网络！"*
  - *"总结一下今天的网络表现，艾特一下其他节点打个招呼。"*
